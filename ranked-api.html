Ranked Api

Linguagem: Java
Framework: Spring Boot
Tecnologias: JPA, H2, Lombok, Devtools e Validation

O projeto é uma API RESTful desenvolvida em Java com o framework Spring Boot. A API é projetada para gerenciar e ranquear itens dentro de listas. Ela oferece uma variedade de funcionalidades, permitindo aos usuários criar, atualizar, deletar e recuperar informações sobre usuários, listas e itens.

Funcionalidades da API:

Usuários: A API possui endpoints para cadastrar, atualizar, deletar e recuperar informações de usuários.

Listas: Os usuários podem criar, atualizar, deletar e recuperar listas. Cada lista pode conter vários itens. Além disso, os usuários podem atualizar a posição de cada lista conforme necessário.

Itens: Dentro de cada lista, os usuários podem criar, atualizar, deletar e recuperar itens. Além disso, os usuários podem atualizar a posição de cada item dentro de uma lista.

Funcionamento:

A API permite que os usuários gerenciem suas listas e os itens dentro delas. Os usuários podem criar novas listas, adicionar itens a essas listas e ranquear os itens conforme desejado. Se um usuário decidir alterar o ranqueamento de um item ou de uma lista, ele pode fazer isso através da API. Da mesma forma, se um usuário quiser remover um item ou uma lista, ele pode fazer isso através da API.
