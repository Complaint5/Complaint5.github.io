Encurtador de Url

Linguagem: Java
Framework: Spring Boot
Tecnologias: JPA, H2, Lombok, Devtools e Validation

Este projeto é uma API RESTful que fornece um serviço de encurtamento de URLs. A API foi desenvolvida usando Java e Spring Boot, e segue os princípios da Clean Architecture, o que facilita a manutenção e a escalabilidade do código.

Funcionalidades Principais:

Cadastro de URL: A API fornece um endpoint para o cadastro de URLs. Os usuários podem enviar uma URL longa, e a API irá gerar e retornar uma URL encurtada correspondente.

Atualização de URL: Os usuários podem atualizar a URL original associada a uma URL encurtada específica através de um endpoint dedicado.

Recuperação de URL: A API permite que os usuários recuperem a URL original a partir do ID da URL cadastrada através de um endpoint GET.

Redirecionamento: Quando a URL encurtada é acessada, a API redireciona o usuário para a URL original correspondente.
