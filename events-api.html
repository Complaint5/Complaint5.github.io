Events Api

Linguagem: Java
Framework: Spring Boot
Tecnologias: JPA, H2, Lombok, Devtools, Validation e Spring Security

Este projeto é uma API RESTful desenvolvida em Java utilizando o framework Spring Boot. A API é projetada para gerenciar o cadastro de eventos estudantis, com endpoints específicos para o cadastro de estudantes, universidades e tipos de usuários.

Os principais componentes da API são:

Estudante: Este endpoint permite o cadastro de estudantes. Ele suporta métodos HTTP como GET (para buscar informações do estudante), POST (para criar um novo estudante), DELETE (para remover um estudante) e UPDATE (para atualizar as informações de um estudante existente).

Universidade: Este endpoint é semelhante ao do estudante, mas é usado para gerenciar as informações das universidades. Ele também suporta os métodos GET, POST, DELETE e UPDATE.

Tipo de Usuário: Este endpoint permite o gerenciamento de diferentes tipos de usuários. Ele suporta os mesmos métodos HTTP que os outros endpoints.

Além disso, a API permite que os usuários sejam buscados com base na instituição e no tipo de cadastro. Isso significa que você pode, por exemplo, buscar todos os estudantes que estão registrados em uma universidade específica ou todos os usuários de um determinado tipo.

A segurança da API é garantida pelo uso do Spring Security com tokens JWT. Isso significa que cada requisição à API deve ser acompanhada de um token JWT válido, que é usado para autenticar o usuário e garantir que ele tem permissão para realizar a ação solicitada.
