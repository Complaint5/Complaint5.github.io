File Api

Linguagem: Java
Framework: Spring Boot
Tecnologias: JPA, H2, Lombok, Devtools e Validation

O projeto é uma API RESTful desenvolvida em Java com o framework Spring Boot. A API é projetada para permitir que os usuários enviem e baixem arquivos por meio de endpoints específicos.

Funcionalidades da API:

Upload de Arquivos: A API possui um endpoint para o upload de arquivos. Os usuários podem enviar arquivos para este endpoint, e a API armazena esses arquivos em um local seguro no servidor.

Download de Arquivos: A API também possui um endpoint para o download de arquivos. Os usuários podem solicitar um arquivo específico por meio deste endpoint, e a API retorna o arquivo solicitado.

Funcionamento:

A API permite que os usuários enviem arquivos para o servidor e baixem arquivos do servidor. Quando um usuário envia um arquivo, a API recebe o arquivo, armazena-o em um local seguro no servidor e retorna uma confirmação de que o arquivo foi recebido e armazenado com sucesso.

Quando um usuário solicita um arquivo, a API verifica se o arquivo solicitado existe no servidor. Se o arquivo existir, a API retorna o arquivo para o usuário. Se o arquivo não existir, a API retorna uma mensagem informando que o arquivo solicitado não foi encontrado.
