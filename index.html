Projeto de envio de emails de bom dia

O projeto tem como objetivo permitir que um usuário cadastre seu e-mail em uma plataforma. Após o cadastro, será enviado um e-mail confirmando que o endereço de e-mail foi cadastrado. Além disso, a plataforma enviará e-mails periódicos com mensagens de bom dia todos os dias.

Projeto de api de cadastro de enventos

O projeto envolve o desenvolvimento de uma API para cadastro de usuários por meio de um front-end. Nesse sistema, o próprio usuário pode inserir seus dados pessoais e endereço. Além disso, a instituição também pode cadastrar informações relevantes, como seus dados e os diferentes tipos de cadastro disponíveis. A categoria do usuário cadastrado será indicada com base nas informações fornecidas.

Desafio Pic Pay

O Desafio Back-End do PicPay consiste em criar uma API que simule um sistema bancário. Nesse sistema, os usuários têm contas e podem realizar transferências entre elas. O objetivo é desenvolver uma solução eficiente e segura para gerenciar essas operações financeiras, garantindo a integridade dos dados e a confiabilidade das transações. A API deve ser capaz de lidar com o cadastro de usuários, consulta de saldos e transferências.

Projeto de uma api para ranqueamento

API voltada para o ranqueamento de itens dentro de listas. A solução oferece endpoints para o cadastro, atualização e exclusão desses itens e listas. Além disso, há a possibilidade de atualizar a posição de cada item e lista, proporcionando uma experiência flexível e eficiente para os usuários.

Projeto para upload e download de aqruivos

O projeto de api para ranquear listas e intes dentre dessas lista com endpoints para cadastro desses itens e listas, atualização, exclusão e atualização da posição do item e lista. Projeto de api para download e upload de arquivos API que permite o download e upload de arquivos. A solução oferece endpoints para gerenciar downloads e uploads dos arquivos. Os usuários podem enviar e baixar arquivos de diversos formatos, incluindo PDFs, imagens e documentos.

Encurtador de Url

API recebe uma URL longa e gera uma versão encurtada e exclusiva. Essa URL reduzida redirecionará os usuários para a URL original quando acessada.
