Bom Dia Api

Linguagem: Java e JavaScript
Framework: Spring Boot
Tecnologias: HTML, CSS, JPA, H2, Lombok, Devtools, Validation e Java Mail

O projeto é um sistema de envio de e-mails diários com mensagens de “Bom dia” para os usuários cadastrados. Ele é composto por duas partes principais: uma API de back-end e uma interface de front-end.

Back-end (API): O back-end, desenvolvido em Java com o framework Spring Boot, gerencia o banco de dados de e-mails dos usuários e programa o envio das mensagens de “Bom dia”. A API possui endpoints para cadastrar um novo e-mail e atualizar a preferência de recebimento de e-mails. O processamento de envio de e-mails é realizado de forma assíncrona, utilizando programação agendada (scheduled).

Front-end: O front-end, desenvolvido com HTML, CSS e JavaScript, permite que os usuários interajam com a API de maneira fácil e intuitiva. Ele oferece formulários para cadastrar um novo e-mail e atualizar a preferência de recebimento de e-mails.

Funcionamento: Todos os dias, a API verifica a lista de e-mails cadastrados e envia uma mensagem de “Bom dia” para cada um deles de forma assíncrona. Se um usuário decidir que não deseja mais receber essas mensagens, ele pode atualizar sua preferência através do front-end, e a API irá parar de enviar e-mails para ele.

Neste projeto, uma vez que um e-mail é cadastrado, ele não pode ser removido da lista de envios. No entanto, o usuário pode optar por não receber mais e-mails atualizando sua preferência.
